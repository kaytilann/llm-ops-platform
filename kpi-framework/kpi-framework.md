# KPI Framework

## North Star Metric
Trusted AI usage at scale.

## Business metrics
- AI cost avoided
- Cost per successful AI task
- AI application adoption
- Time to launch new AI workflow
- AI incident reduction

## AI quality metrics
- Eval pass rate
- Groundedness score
- Tool-use accuracy
- Hallucination rate
- User feedback score
- Escalation accuracy

## Cost and performance metrics
- Total model spend
- Cost per request
- Cost per application
- Cache hit rate
- Token compression rate
- P50/P95/P99 latency
- Provider fallback rate

## Governance metrics
- Policy violation rate
- PII redaction events
- Prompt injection blocks
- Human review rate
- Audit completeness
- Model promotion compliance
